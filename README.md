# 📚 Documentación Técnica - SoyITPro

Repositorio con **guías técnicas paso a paso** y documentos de procedimientos para administración de sistemas, configuración de infraestructura y solución de problemas.

## 📖 Contenido

Este repositorio está organizado por año y contiene guías prácticas sobre:

- **Virtualización y GPU**: Passthrough de GPU en máquinas virtuales
- **Containerización**: Docker en WSL y Windows 11
- **Seguridad**: Certificados RDP autofirmados, configuración de seguridad
- **Configuración del Sistema**: Active Directory, gpedit en Windows Home, ISO personalizadas
- **Actualizaciones y Parches**: Notificaciones RDP y actualizaciones críticas
